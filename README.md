# Project Aurora DEX Assistant — UI Redesign

Mobile-first Project Aurora web app redesigned to feel like a simple warehouse operations assistant rather than a database.

## What changed

- Maersk-inspired portal/login screen.
- Role-based workflow for Warehouse Associate, Warehouse Lead, Manager / Supervisor, and Admin.
- Admin area requires an extra password: `Nacuchis1`.
- Dashboard tile navigation similar to an iPad app.
- Guided DEX observation form with large buttons and minimal typing.
- Live reconcile meter with Unclassified / Unlogged Gap support.
- Locked 6/10 baseline remains seeded.
- Planned break is separated from restart overrun.
- Active speed is only the log's own observed cases/min.
- Reports, Pareto, Savings, Export, JSON backup, and PWA/offline support.

## Deploy to GitHub Pages

Upload every file in this folder to the root of the GitHub repository and keep Pages set to `main` / `(root)`.

## Important

This is still device-local storage. Data entered on a phone will not automatically appear on a laptop unless Stephanie exports/imports JSON or CSV.
